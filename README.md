# Ride-Booking-System-DSA-project-


A console-based ride-booking application built in C++ using fundamental Data Structures and Algorithms.  
The system simulates a ride-hailing service for different locations in Islamabad.

---

## 🚀 Features

- 👤 User Management: Sign up and login system
- 🗺️ City Graph: Graph-based model of Islamabad zones and connections
- 📍 Shortest Path: Dijkstra’s Algorithm for optimal route selection
- 🚗 Driver Matching: Priority-based driver selection (distance + rating + vehicle type)
- 💰 Fare Calculation: Dynamic pricing based on distance and ride type
- 📜 Ride History: Tracks rides during program execution
- 🌳 Zone Search: BST-based searching and sorting of locations

---

## 🧠 Data Structures Used

| Data Structure | Purpose |
|----------------|---------|
| Graph (Adjacency List) | City representation |
| Dijkstra’s Algorithm | Shortest path calculation |
| Binary Search Tree (BST) | Zone management and searching |
| Priority Queue (Heap) | Driver ranking system |
| Unordered Map | Fast lookup for users and drivers |
| Vector / Linked List | Ride history storage |
